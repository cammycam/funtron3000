BUILD INSTUCTIONS:

Install Node.js, along with Grunt and the required plugins.
Once Node.js is installed, Grunt and its plugins can be installed by the command:

npm install

Once you have the tools you need installed issues these commands to build the core js files.

grunt concat
grunt uglify

For More information on Grunt Builds visit: http://gruntjs.com/getting-started
